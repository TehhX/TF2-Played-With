# Contributing

Contributions are welcome, just submit issues or PRs. If they're good, I'll add/address them. If they suck, I'll try to refactor them or something. Either way, don't be afraid to contribute, half-solutions are welcome.

## Issue solving

If you see an issue you want to solve, request assignment. After receiving assignment of the issue, take a crack at it and submit a PR when finished.

## Artifical Intelligence

AI generated code, issues, PRs, etc. are not welcome in this repository in any form.
